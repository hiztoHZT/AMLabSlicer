# AMLabSlicer

一款基于 WPF 开发的 3D 打印切片与可视化上位机软件，致力于做一个完全通用的切片软件，特别针对多轴进行开发。
还没开发完成

## 🤔 预想功能
* 改变切片软件逻辑，对于切片参数的设置，从打印机-线材-工艺，改为算法-打印机-线材-工艺，针对不同算法适配不同参数输入内容，满足各类切片方法的导入。
* 重构切片预览窗口，针对多轴打印中不断变化的LPD，不仅可以同其他切片软件一样可以逐层查看，也可以对不同LPD进行分组，查看更加详细的参数变化，客观的预览复杂的五轴Gcode代码。
